📊 Student Productivity Tracker

A modern front-end web application designed to help students manage
daily tasks, track progress, and improve productivity. This project was
built as part of a Hackathon Level-2: Interactivity Implementation,
focusing on dynamic UI updates using JavaScript.

🚀 Features ✅ Add new tasks with subject, priority, and due date ✅
Mark tasks as completed / pending ✅ Edit and delete tasks ✅ Filter
tasks (All / Completed / Pending) ✅ Real-time progress tracking ✅
Animated progress bar ✅ Overdue task indicator ✅ Data persistence
using LocalStorage 🧠 Concepts Used DOM Manipulation Event Handling
State Management (JavaScript array) LocalStorage for data persistence
Dynamic UI rendering Basic UI/UX design principles 🛠️ Tech Stack HTML5
-- Structure CSS3 -- Styling & animations JavaScript (Vanilla) -- Logic
& interactivity 📂 Project Structure 📁 Student-Productivity-Tracker │──
index.html │── style.css │── script.js │── README.md ⚙️ How to Run
Download or clone the repository Open index.html in your browser Start
adding tasks and track your progress 🎯 Hackathon Objectives Covered

✔ Display dynamic activity list ✔ Update task status (Pending →
Completed) ✔ Real-time progress calculation ✔ No page reload (dynamic
updates) ✔ Use of LocalStorage (optional requirement) ✔ Clean separation
of HTML, CSS, and JavaScript
